# SR Fitness

SR Fitness is a premium responsive Gym & Fitness website built using HTML, CSS, and JavaScript.

## Features

- Modern black-and-white fitness theme
- Home page with premium hero section
- Workout plans
- Diet plans
- BMI calculator
- Trainer profiles
- Membership pricing
- Login and create account using localStorage
- Responsive design
- Interactive buttons and smooth UI

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage

## Project Status

Frontend completed. Backend can be added later using Node.js, Express.js, and MongoDB.
